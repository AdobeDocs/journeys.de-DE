# upper {#upper}

Gibt eine Großbuchstabenversion des Parameters zurück.

## Kategorie

String

## Funktionssyntax

`upper(<parameters>)`

## Unterschrift und zurückgegebener Typ

`upper(<string>)`

Gibt eine Zeichenfolge zurück.

## Beispiel

`upper("b")`

Gibt &quot;B&quot;zurück.
