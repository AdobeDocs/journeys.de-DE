# length {#length}

Gibt die Anzahl der Zeichen des Zeichenfolgenausdrucks im Parameter zurück.

## Kategorie

Zeichenfolge

## Funktionssyntax

`length(<parameters>)`

## Parameter

* string

## Signatur und zurückgegebener Typ

`length(<string>)`

Gibt eine Ganzzahl zurück.

## Beispiel

`length("Hello World")`

Gibt 11 zurück.
