# isNotEmpty {#isNotEmpty}

Gibt „true“ zurück, wenn die Zeichenfolge im Parameter nicht leer ist.

## Kategorie

Zeichenfolge

## Funktionssyntax

`isNotEmpty(<parameters>)`

## Parameter

* string

## Signatur und zurückgegebener Typ

`isNotEmpty(<string>)`

Gibt einen booleschen Wert zurück.

## Beispiel

`isNotEmpty("")`

Gibt „false“ zurück.

`isNotEmpty("hello")`

Gibt „true“ zurück.
